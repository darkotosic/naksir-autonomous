Naksir Autonomous – Static Tickets Feed

GitHub Pages endpoint used by Android apps and integrations.

Endpoints

JSON feeds are available at:

  • /naksir-autonomous/tickets.json – daily tickets
  • /naksir-autonomous/evaluation.json – ticket results
  • /naksir-autonomous/btts.json – BTTS Yes daily feed
  • /naksir-autonomous/btts.stats.json – BTTS fixture-level stats

Example links on this site:

  • ./tickets.json
  • ./evaluation.json
  • ./btts.json
  • ./btts.stats.json
Debug

If you see HTTP 404 for tickets.json or evaluation.json, check that the GitHub Actions workflow finished successfully and copied files from outputs/ into this public/ folder.
